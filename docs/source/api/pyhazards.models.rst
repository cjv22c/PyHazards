pyhazards.models package
========================

Model Category
--------------

CNN-ASPP
~~~~~~~~

:doc:`CNN-ASPP </modules/models_wildfire_aspp>`

Explainable CNN-ASPP model for next-day wildfire spread prediction.

DNN-LSTM-AutoEncoder
~~~~~~~~~~~~~~~~~~~~

:doc:`DNN-LSTM-AutoEncoder </modules/models_wildfire_fpa>`

Two-stage FPA-FOD framework with a DNN stage for cause/size prediction and an
LSTM + autoencoder stage for weekly wildfire forecasting.

HydroGraphNet
~~~~~~~~~~~~~

:doc:`HydroGraphNet </modules/models_hydrographnet>`

Physics-informed graph neural network for flood forecasting on irregular
meshes.
